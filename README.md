# Exercícios - Mundo 2 Python3

Este repositório contém a resolução dos exercícios propostos no **Mundo 2** do curso de Python3 do professor **Gustavo Guanabara**, disponível no [Curso em Vídeo](https://www.cursoemvideo.com/).

O foco dessa etapa é aprofundar conhecimentos em **estruturas condicionais** e **laços de repetição**.

## Conteúdos Abordados

- Condições (`if`, `elif`, `else`)
- Laços `for` e `while`
- Trabalhando com `range()` e incrementos
- `break` e controle de fluxo
- Interação com o `input()`
- Cálculos e lógica de programação
- Mini projetos simples

Cada exercício está salvo em um arquivo separado, nomeado conforme a numeração e nome do exercício.
